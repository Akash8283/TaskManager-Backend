# Task Manager Backend

Backend service for a simple Task Manager application built using Node.js, Express, and MongoDB.

## Tech Stack
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose

## Features
- Add a task
- Get all tasks
- Delete a task

## API Endpoints
- POST /tasks – Add a new task
- GET /tasks – Get all tasks
- DELETE /tasks/:id – Delete a task by ID

## Setup
1. npm install
2. Add MongoDB connection string in environment variables
3. npm run dev
